# Portfolio JTBD

## User persona

- **Who:** Recruiters and hiring managers evaluating internship candidates.
- **Goal:** Decide quickly whether Sheharyar fits a software engineering, AI/ML, or data engineering internship.
- **Context:** Usually a short desktop review from a resume, LinkedIn profile, or application.
- **Success metric:** They can find fit, evidence, and a contact path in under two minutes.

## Job statement

When I am screening internship candidates, I want to see a candidate's target roles, credible technical evidence, and resume immediately, so I can confidently decide whether to start a conversation.

## Current solution and pain points

- A generic landing page hides qualifications below the fold.
- A recruiter must hunt for projects, technical depth, and contact details.
- Unclear availability or target roles makes prioritization difficult.
